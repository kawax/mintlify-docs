## Overview

[revolution/laravel-amazon-bedrock](https://github.com/invokable/laravel-amazon-bedrock) is a driver that lets you use Amazon Bedrock through the [Laravel AI SDK](https://laravel.com/docs/13.x/ai-sdk). You can access multiple Bedrock models through one Laravel AI SDK interface.

The main features are as follows.

| Feature | Available with Bedrock API key only | Supported models / services |
| --- | --- | --- |
| Text generation / streaming | ✅ | Anthropic Claude, Amazon Nova, Meta Llama, Mistral, Cohere Command R, DeepSeek, AI21 Jamba |
| Tool use (Function Calling) | ✅ | Anthropic Claude, Amazon Nova, Meta Llama 3.1+, Mistral Large, Cohere Command R |
| Structured output | ✅ | Anthropic Claude, Amazon Nova, Meta Llama 3.1+, Mistral Large, Cohere Command R |
| Image generation | ✅ | Stability AI (default) |
| Audio (TTS) | ❌ | Amazon Polly |
| Transcription (STT) | ❌ | Amazon Nova 2 Lite (Converse API AudioBlock) |
| Embeddings | ✅ | Amazon Titan Embeddings V2 (default), Cohere Embed V3 / V4 |
| Reranking | ❌ | Cohere Rerank 3.5, Amazon Rerank 1.0 |

<Info>
  `❌` in this table means the feature is not available with only a Bedrock API key. It does not mean the feature itself is unsupported.
</Info>

<Info>
  You can choose from three authentication methods: Bedrock API key, AWS IAM credentials (SigV4), and the default AWS credential chain (IAM roles and related sources).
</Info>

## Configuration

Add the `bedrock` provider to `config/ai.php`. Switch your default providers to Bedrock if needed.

```php
'default' => 'bedrock',
'default_for_images' => 'bedrock',
'default_for_audio' => 'bedrock',
'default_for_embeddings' => 'bedrock',
'default_for_reranking' => 'bedrock',
'default_for_transcription' => 'bedrock',
```

### Option 1: Bedrock API key

```php
'providers' => [
    'bedrock' => [
        'driver' => 'bedrock',
        'key' => env('AWS_BEDROCK_API_KEY', ''),
        'region' => env('AWS_DEFAULT_REGION', 'us-east-1'),
    ],
],
```

```dotenv
AWS_BEDROCK_API_KEY=your_api_key
AWS_DEFAULT_REGION=us-east-1
```

<Warning>
  A Bedrock API key only works for the Bedrock Runtime API. It does not work with reranking (`bedrock-agent-runtime`) or Amazon Polly (TTS). Use SigV4 or the default AWS credential chain for those features.
</Warning>

### Option 2: AWS IAM credentials (SigV4)

```php
'providers' => [
    'bedrock' => [
        'driver' => 'bedrock',
        'key' => env('AWS_ACCESS_KEY_ID'),
        'secret' => env('AWS_SECRET_ACCESS_KEY'),
        'token' => env('AWS_SESSION_TOKEN'),
        'region' => env('AWS_DEFAULT_REGION', 'us-east-1'),
    ],
],
```

```dotenv
AWS_ACCESS_KEY_ID=AKIA...
AWS_SECRET_ACCESS_KEY=...
AWS_SESSION_TOKEN= # optional
AWS_DEFAULT_REGION=us-east-1
```

<Info>
  Set `AWS_SESSION_TOKEN` only when you use temporary credentials (STS).
</Info>

### Option 3: Default AWS credential chain (IAM roles)

```php
'providers' => [
    'bedrock' => [
        'driver' => 'bedrock',
        'region' => env('AWS_DEFAULT_REGION', 'us-east-1'),
    ],
],
```

```dotenv
AWS_DEFAULT_REGION=us-east-1
```

<Info>
  This option is usually safest in environments such as EC2, ECS, and Lambda where IAM roles are available.
</Info>

## Basic usage

```php
use App\Ai\Agents\BedrockAgent;
use Laravel\Ai\Contracts\Agent;
use Laravel\Ai\Promptable;

class BedrockAgent implements Agent
{
    use Promptable;

    public function instructions(): string
    {
        return 'You are an expert in software development.';
    }
}

$response = (new BedrockAgent)->prompt('Summarize what is new in Laravel 13.');

echo $response->text;
```

## Streaming

You can use Bedrock streaming through the Laravel AI SDK.

```php
use App\Ai\Agents\BedrockAgent;
use Illuminate\Support\Facades\Route;

Route::get('/stream', function () {
    return (new BedrockAgent)->stream('Tell me about Laravel 13.');
});
```

<Tip>
  Bedrock streaming is not available in the official AWS SDK for PHP or Prism. This package provides a practical implementation for PHP and Laravel.
</Tip>

## Tool use (Function Calling)

```php
use Illuminate\Contracts\JsonSchema\JsonSchema;
use Laravel\Ai\Contracts\Tool;
use Laravel\Ai\Tools\Request;

class GetWeather implements Tool
{
    public function description(): string
    {
        return 'Return the current weather for a city.';
    }

    public function schema(JsonSchema $schema): array
    {
        return [
            'city' => $schema->string()->required(),
        ];
    }

    public function handle(Request $request): string
    {
        return json_encode(['city' => $request['city'], 'condition' => 'sunny']);
    }
}
```

```php
use Laravel\Ai\Attributes\MaxSteps;
use Laravel\Ai\Contracts\Agent;
use Laravel\Ai\Promptable;

#[MaxSteps(5)]
class WeatherAgent implements Agent
{
    use Promptable;

    public function tools(): array
    {
        return [new GetWeather];
    }

    public function instructions(): string
    {
        return 'You are a weather assistant.';
    }
}

$response = (new WeatherAgent)->prompt('What is the weather in Tokyo?');
echo $response->text;
```

## Conversation history

Implement the `Conversational` interface to include previous messages in each prompt.

```php
use App\Models\ChatHistory;
use Laravel\Ai\Contracts\Agent;
use Laravel\Ai\Contracts\Conversational;
use Laravel\Ai\Messages\Message;
use Laravel\Ai\Promptable;

class ChatAgent implements Agent, Conversational
{
    use Promptable;

    public function __construct(public int $userId) {}

    public function instructions(): string
    {
        return 'You are a helpful assistant.';
    }

    public function messages(): iterable
    {
        return ChatHistory::where('user_id', $this->userId)
            ->latest()
            ->limit(20)
            ->get()
            ->reverse()
            ->map(fn ($message) => new Message($message->role, $message->content))
            ->all();
    }
}

$response = (new ChatAgent(auth()->id()))->prompt('Summarize our previous conversation if we had one.');
echo $response->text;
```

## Structured output

```php
use Illuminate\Contracts\JsonSchema\JsonSchema;
use Laravel\Ai\Contracts\Agent;
use Laravel\Ai\Contracts\HasStructuredOutput;
use Laravel\Ai\Promptable;

class ExtractPerson implements Agent, HasStructuredOutput
{
    use Promptable;

    public function instructions(): string
    {
        return 'Extract person information from the text.';
    }

    public function schema(JsonSchema $schema): array
    {
        return [
            'name' => $schema->string(),
            'age' => $schema->integer(),
        ];
    }
}

$response = (new ExtractPerson)->prompt('Alice is 25 years old.');
echo $response['name'];
```

## Image generation

Example using Stability AI.

```php
use Laravel\Ai\Image;

$response = Image::of('A cat walking under cherry blossoms')
    ->square()
    ->generate(provider: 'bedrock');

$response->store('images', 's3');
```

## Audio (TTS)

Example using Amazon Polly.

```php
use Laravel\Ai\Audio;

$response = Audio::of('Generate speech with Laravel.')
    ->voice('Joanna')
    ->generate(provider: 'bedrock');

$path = $response->storeAs('voice.mp3');
```

## Transcription (STT)

Example using Nova 2 Lite.

```php
use Laravel\Ai\Transcription;

$response = Transcription::fromPath('/path/to/audio.mp3')
    ->generate(provider: 'bedrock');

echo $response->text;
```

## Embeddings

```php
use Laravel\Ai\Embeddings;

$response = Embeddings::for(['Laravel', 'Bedrock'])
    ->dimensions(1024)
    ->generate(provider: 'bedrock', model: 'amazon.titan-embed-text-v2:0');

$vector = $response->first();
```

## Reranking

```php
use Laravel\Ai\Reranking;

$response = Reranking::of([
    'Laravel is a PHP framework.',
    'Bedrock provides foundation models.',
    'Laravel AI SDK provides a unified API.',
])->limit(2)->rerank(
    query: 'Explain Laravel and AI',
    provider: 'bedrock'
);

echo $response->first()->document;
```
